# Ghost Analytics

Ghost Analytics is the parent brand for sport-specific prediction tools powered by GhostIQ.

Initial product:

- **MLB:** BarrelIQ, a daily home run target and matchup intelligence tool.

Future products:

- **NBA:** CourtIQ
- **NFL:** RedZoneIQ

The goal is to build one real product platform, not a disposable dashboard.

## Product Standard

Ghost Analytics should feel:

- Professional
- Fast to scan
- Data-first
- Visually distinctive
- Mobile usable
- Honest about uncertainty

The site should never imply guaranteed outcomes. It should show edge, confidence, volatility, and why a play is being flagged.

## Core Rules

- Website time zone: **Eastern Time**
- Public dashboard labels should be clear and explainable.
- No temporary one-day labels such as `Source Sheet`.
- Model data can learn from prior manual sheets, but those sheets should not appear as public-facing dashboard labels.
- Confirmed lineups and remaining-game filtering matter.
- Pairings must use dedicated logic and should not simply combine the highest-ranked hitters.
- Audits must be saved every day, even when the API fails, so failures are visible instead of silent.
